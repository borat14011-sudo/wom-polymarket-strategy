# STRATEGIC DEBATE FRAMEWORK
## 4-Agent Synthesis Process

### AGENT ASSIGNMENTS

**Agent 1: Market Analysis**
- Identify best opportunities
- Volume/liquidity analysis
- Edge identification

**Agent 2: Portfolio Construction**  
- Position sizing (Kelly)
- Correlation analysis
- Risk management rules

**Agent 3: Strategy Validation**
- Pattern recognition
- Historical backtesting
- Proven edge identification

**Agent 4: Execution & Timing**
- When to trade
- Order types
- Cost minimization

### SYNTHESIS PROCESS

1. **Individual Research** (20 min)
   - Each agent conducts deep analysis
   - Spawn subagents as needed
   - Gather data/evidence

2. **Cross-Examination** (10 min)
   - Agents review each other's findings
   - Identify conflicts/gaps
   - Challenge assumptions

3. **Synthesis** (10 min)
   - Merge best insights from all agents
   - Resolve conflicts
   - Create unified framework

4. **Presentation** (10 min)
   - Consolidate into investor-grade deck
   - Actionable bet recommendations
   - Risk disclosure

### OUTPUT DOCUMENTS

1. EXECUTIVE_SUMMARY.md - Key findings
2. OPPORTUNITY_MATRIX.md - Ranked trades
3. PORTFOLIO_BLUEPRINT.md - Allocation strategy
4. ACTION_PLAN.md - Immediate next steps
5. MASTER_PRESENTATION.md - Complete synthesis

### SUCCESS METRICS

- 10+ actionable trade ideas
- Risk-adjusted returns >15%
- Clear entry/exit rules
- Portfolio diversification
- Execution playbook
