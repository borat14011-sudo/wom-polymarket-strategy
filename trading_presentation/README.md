## ğŸš€ How to Build the Canva Presentation

### Step 1: Set Up Your Canva Project
1. Go to [Canva](https://www.canva.com)
2. Create new presentation (16:9 ratio)
3. Set color palette:
   - Primary: #1a365d (Navy)
   - Secondary: #d4af37 (Gold)
   - Accent: #319795 (Teal)
4. Set fonts: Montserrat (headers), Inter (body)

### Step 2: Create Slides (25-30 total)
Follow `CANVA_SLIDE_GUIDE.md` for detailed slide-by-slide instructions including:
- Slide types and layouts
- Specific content for each slide
- Chart specifications
- Visual style guidelines

### Step 3: Import Data
For charts, you have 3 options:
1. **Direct CSV Import:** Upload CSV files from `/data` folder to Canva charts
2. **Copy-Paste:** Open CSV in Excel, copy data, paste into Canva chart
3. **Create Externally:** Build charts in Excel/Google Sheets, screenshot, upload as image

### Step 4: Add Content
- Copy text from `ALPHA_FLOW_PRESENTATION_CONTENT.md`
- Use `SPEAKER_NOTES.md` for presenter notes
- Ensure all key numbers match the data files
