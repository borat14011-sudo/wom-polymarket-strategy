API ENDPOINTS
================================================================================

GET /api/status           → System overview and database stats
GET /api/markets          → List all tracked markets
GET /api/markets/<id>     → Detailed market data with history
GET /api/signals          → Trading signals with performance
GET /api/performance      → Performance metrics and equity curve
GET /api/top-hype         → Top markets by hype score

All endpoints return JSON data.

USAGE TIPS
================================================================================

1. Run data collectors BEFORE starting dashboard

   - More data = better visualizations

2. Keep API running continuously
   - Dashboard polls API every 60 seconds
   - Stop with Ctrl+C

3. Open dashboard in modern browser
   - Chrome, Firefox, Safari, Edge (latest versions)
   - JavaScript must be enabled

4. Monitor connection status
   - Green dot = connected
   - Red dot = disconnected (check if API is running)

5. Click markets for detailed views
   - Shows price history chart
   - Top tweets by engagement
   - Market metadata

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "Database not found"
Solution: Run data collectors to create polymarket_data.db

Problem: "Disconnected" status in dashboard
Solution: Start API server with "python api.py"

Problem: Empty tables in dashboard
Solution: Wait for data collectors to run (15+ minutes)

Problem: Charts not rendering
Solution: Check internet connection (Chart.js loads from CDN)

Problem: Port 5000 already in use
Solution: Kill process or change port in api.py and dashboard.html

See DASHBOARD-README.md for complete troubleshooting guide.
