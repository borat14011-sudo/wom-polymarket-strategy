## Phase 4: Live Betting Protocol

### Live Betting Rules

**When to Bet Live**:
1. End of Q1 if score within 7 points
2. Halftime if underdog trailing by <14
3. End of Q3 if game within 10 points
4. 2-minute warning if comeback possible

**Live Bet Sizing**:
- Standard: 2% of original bankroll
- Aggressive (clear value): 4% of original bankroll
- Max single live bet: 5% of original bankroll

**Live Markets to Target**:
1. Quarter winner (underdog in Q2 or Q4)
2. Next score method (pass/run/FG)
3. Player to score next TD
4. Total points over/under adjustments

### Live Betting Flowchart

```
Q1 END - Score Within 7?
├─ YES → Bet Underdog to Win Q2 (2%)
├─ NO (favorite up big) → Skip

HALFTIME - Underdog Losing by <14?
├─ YES → Add to Underdog ML if price >0.25 (5%)
├─ NO → Hold position

Q3 END - Game Within 10?
├─ YES → Bet Underdog Q4 Winner (2%)
├─ NO → Consider hedge if winning big

2-MIN WARNING - Underdog Has Ball?
├─ YES → Bet Underdog to Score (1%)
├─ NO → Skip
```

## Phase 5: Risk Management Protocol

### Daily Checklist

**Morning of Game**:
```
□ Review all positions
□ Check for overnight news
□ Verify no new injuries
□ Confirm bankroll allocation matches plan
```

**If Winning Big (+50% bankroll)**:
```
□ Take 25% profit on biggest winner
□ Let remainder ride
□ DO NOT increase bet sizes
□ Consider hedge on opposing team if >+100%
```

**If Losing Big (-30% bankroll)**:
```
□ STOP all new bets
□ Review remaining live reserve (15%)
□ Only bet if CLEAR value opportunity
□ Do not chase losses
```

**If Losing Huge (-50% bankroll)**:
```
□ Close all positions immediately
□ Accept the loss
□ Do not try to "get it back"
□ Document lessons learned
```
